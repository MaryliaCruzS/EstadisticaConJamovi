Estadística para el Sector Público con Jamovi

Capítulo 2 Introducción

2.1 ¿Por qué Jamovi para el sector público?

Jamovi es un programa gratuito para realizar cálculos estadísticos, esto facilita el proceso de adquisión de licencias de otros softwares estadísticos de paga.

Es un software de fácil uso, debido a que desde su creación se consideró la versatilidad del SPSS, uno de los softwares de pago más empleados. Sin embargo, Jamovi es una interfaz gráfica de R, un software muy pontente basado en el lenguaje de objetos.

Jamovi también nos permitirá realizar un reporte en pdf y guardar nuestro nuevos avances como un proyecto de Jamovi.

2.2 ¿Cómo instalar JAMOVI?

Enel siguiente link https://www.jamovi.org/download.html puede encontrar el instalador para Windons y Mac. Se recomienda instalar la versión SOLID, la más recomendada para la mayoría de usuarios.
